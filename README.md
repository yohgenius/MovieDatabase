# MovieDatabase
TMDB project

### **Features**

- See genre list as grid list on homepage, to narrow down the list
- See available movie list according to genre
- See movie detail
- See movie review
- Watch movie trailer (opening external app)
- Favorite screen

### **Libraries**

- dagger-hilt
- coroutines
- retrofit
- room
- livedata
- tmdb api
